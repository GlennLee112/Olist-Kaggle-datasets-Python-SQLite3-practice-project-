# Olist Kaggle datasets (Data analysis practice project)
Practice project to improve general data analysis skills, as well as general task automation utilizing Python, including automated data querying app with UI.

Data sample from: https://www.kaggle.com/datasets/terencicp/e-commerce-dataset-by-olist-as-an-sqlite-database

Create a "Dataset" folder and paste the data from link above to run test query (sqlite dataset not included due to file size)

## Practice file structure
### **Py_Practice**

**1. Py_Practcie-Tkinter_main**

Automated SQL Querying app using Tkinter built-in Python module for creating a UI for used to select date range and output file to the **Output** folder to be read on the next step for Python EDA.

**2. Py_Practice-Python_EDA**

Exploratory Data Analysis (EDA) within Python environment
